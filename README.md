Popis projektu
Táto aplikácia umožňuje generovať N náhodných čísel, ktoré sa následne upravia podľa zadanej hodnoty A a zobrazia sa v grafe. Používateľ zadáva vstupné hodnoty cez jednoduché grafické rozhranie (GUI).

Funkcionalita
- Generovanie N náhodných čísel
- Výpočet súčtu a súčinu pôvodných čísel
- Pripočítanie hodnoty A ku každému číslu
- Zobrazenie upravených čísel v grafe
- GUI obsahuje:
- 3 štítky (labely) zobrazujúce výsledky
- Tlačidlo „Nové čísla“ na spustenie výpočtu
- Vstupné polia pre N a A

Použitie
- Zadajte počet čísel N a hodnotu A.
- Kliknite na tlačidlo „Nové čísla“.
- Aplikácia zobrazí:
- Súčet pôvodných čísel
- Súčin pôvodných čísel
- Graf upravených čísel (x + A)
